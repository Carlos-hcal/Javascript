# Números

JavaScript tem **apenas um tipo de números** – ponto flutuante de 64 bits. É o mesmo que o `duplo` do Java. Ao contrário da maioria das outras linguagens de programação, não existe um tipo inteiro separado, portanto 1 e 1,0 têm o mesmo valor.

Neste capítulo, aprenderemos como criar números e realizar operações com eles (como adições e subtrações).
