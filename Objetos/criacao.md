# Criação

Existem duas maneiras de criar um `objeto` em JavaScript:

1. literalmente

    ```js
    objeto var = {};
    // Sim, simplesmente um par de aparelho!
    ```

    > **_Nota:_** esta é a forma **recomendada**.

2. e orientado a objetos

    ```js
    var objeto = new Objeto();
    ```

    > **_Nota:_** é quase como Java.
